# 한국 국가공인 자격증 종류 정리

한국의 자격증은 크게 **국가기술자격, 국가전문자격, 국가공인 민간자격** 세 가지로 구분된다. 시행기관과 근거 법령이 각각 다르므로 일정 데이터 수집 시 별도 파이프라인 설계가 필요하다.

---

## 1. 국가기술자격 (National Technical Qualifications)

근거 법령: 국가기술자격법
주관: 고용노동부 / 시행: 한국산업인력공단(Q-Net), 대한상공회의소, 한국방송통신전파진흥원 등
등급 체계: 기술사 → 기능장 → 기사 → 산업기사 → 기능사 → (서비스분야 1·2·3급)

대표 종목 (총 500여 종목):

| 분야 | 대표 자격증 |
|---|---|
| 정보통신/IT | 정보처리기사·산업기사, 정보보안기사, 빅데이터분석기사, 정보통신기사, 컴퓨터활용능력 1·2급, 워드프로세서 |
| 건축/토목 | 건축기사, 토목기사, 건축설비기사, 조경기사, 측량 및 지형공간정보기사 |
| 기계/금속 | 일반기계기사, 건설기계설비기사, 용접기사, 금속재료기사 |
| 전기/전자 | 전기기사·산업기사·기능사, 전자기사, 소방설비기사(전기/기계) |
| 화공/환경 | 화공기사, 대기환경기사, 수질환경기사, 폐기물처리기사 |
| 안전관리 | 산업안전기사, 건설안전기사, 가스기사, 소방안전관리자 |
| 식품/조리 | 조주기능사, 한식·양식·중식·일식조리기능사, 제과·제빵기능사, 식품기사 |
| 서비스 | 직업상담사 1·2급, 사회조사분석사, 컨벤션기획사, 텔레마케팅관리사 |

---

## 2. 국가전문자격 (National Professional Qualifications)

근거 법령: 각 부처 개별법 (의료법, 변호사법, 공인회계사법 등)
주관: 각 부처(보건복지부, 법무부, 금융위 등) / 시행: 한국산업인력공단, 금융감독원, 한국부동산원 등

| 분야 | 대표 자격증 |
|---|---|
| 의료/보건 | 의사·치과의사·한의사 국가시험, 약사, 간호사, 간호조무사, 물리치료사, 임상병리사, 방사선사, 영양사, 위생사 |
| 법률 | 변호사시험, 법무사, 변리사, 행정사, 공인노무사 |
| 회계/세무 | 공인회계사(CPA), 세무사 |
| 부동산/감정 | 공인중개사, 감정평가사, 주택관리사보 |
| 금융/보험 | 보험계리사, 손해사정사, 보험중개사 |
| 건축/기술 | 건축사, 기술사(분야별) |
| 노무/복지 | 공인노무사, 사회복지사 1급, 직업능력개발훈련교사 |
| 무역/물류 | 관세사, 물류관리사, 국제무역사 |
| 교육 | 평생교육사, 한국어교원, 청소년상담사·지도사 |
| 기타 | 소방시설관리사, 군무원, 경비지도사 |

## 3. 국가공인 민간자격 (Nationally Recognized Private Qualifications)

근거 법령: 자격기본법
주관: 한국직업능력연구원 / 시행: 각 민간 발급기관 (총 100여 종목)

| 분야 | 대표 자격증 |
|---|---|
| 한국사 | 한국사능력검정시험 (심화·기본) |
| 어학 | TOEIC, TOEFL, TEPS, OPIc, JLPT, HSK, KBS한국어능력시험 |
| 금융/투자 | 투자자산운용사, 펀드투자권유자문인력, 파생상품투자권유자문인력, 증권투자권유자문인력, AFPK, CFP, 신용분석사, 자산관리사(FP) |
| 회계 | 전산회계 1·2급, 전산세무 1·2급, FAT, TAT, ERP정보관리사 |
| IT/OA | ITQ, MOS, 리눅스마스터, SQLD/SQLP, ADsP/ADP, 정보보안전문가 |
| 한자/국어 | 한자능력검정시험, 실용한자, 한국실용글쓰기검정 |
| 무역/사무 | 무역영어, 비서, 한국실무능력검정 |
| 디자인/기타 | GTQ(그래픽기술자격), 텔레마케팅, 브레인트레이너 |

## 4. 일정 데이터 수집 관점에서의 시행기관 분류

서비스 구축 시 크롤링/연동해야 할 주요 사이트:

- **Q-Net (한국산업인력공단)** — 국가기술자격 대부분 + 국가전문자격 다수
- **대한상공회의소 자격평가사업단** — 컴활, 워드, 한자, 전산회계 등
- **한국방송통신전파진흥원(KCA)** — 정보통신·전파 관련
- **금융감독원 / 금융투자협회** — 보험계리사, 투자자산운용사 등
- **국사편찬위원회** — 한국사능력검정시험
- **YBM / ETS / OPIc 등** — 어학 시험별 개별 사이트
- **각 협회 (대한건축사협회 등)** — 건축사 등 일부 전문자격

---

## 5. 자격증 메타데이터 표준 필드 (PRD 설계 입력값)

각 시험 회차마다 수집·표준화해야 할 필드:

- 자격증명 (한글/영문), 카테고리, 등급, 시행기관, 근거법령
- 시험 회차 번호, 시험 연도
- 원서접수 시작/마감일시, 추가접수 기간
- 시험 시행일 (필기/실기/면접 구분)
- 합격자 발표일
- 응시료, 시험과목, 합격기준
- 공고 원문 URL, 접수 URL
